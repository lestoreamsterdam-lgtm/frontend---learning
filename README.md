# 🎓 Frontend Learning - UI/UX, CSS, HTML & JavaScript

Repositório para organizar meus estudos de desenvolvimento frontend, cobrindo os fundamentos essenciais para criar interfaces web modernas e responsivas.

## 📚 Estrutura de Estudos

### 🏗️ HTML
- Estrutura semântica
- Forms e validação
- Acessibilidade
- Meta tags e SEO

### 🎨 CSS
- Flexbox e Grid
- Responsividade
- Animações
- Pré-processadores (SASS/LESS)
- Frameworks (Bootstrap, Tailwind)

### ⚡ JavaScript
- Fundamentos (variáveis, funções, loops)
- DOM manipulation
- Eventos
- Async/Await e Promises
- ES6+ features
- Bibliotecas (React, Vue)

### 🎭 UI/UX
- Design principles
- Wireframing
- Prototipagem
- User research
- Accessibility (WCAG)
- Design systems
