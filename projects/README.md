# 🚀 Projetos Integrados

Projetos que combinam múltiplos conceitos: HTML, CSS, JavaScript e princípios UI/UX.

## 📂 Estrutura

```
projects/
├── README.md
├── beginner/
│   ├── portfolio-page/
│   ├── todo-app/
│   └── calculator/
├── intermediate/
│   ├── e-commerce-product-page/
│   ├── weather-app/

└── advanced/
    ├── social-media-clone/
    ├── real-time-chat/
    └── portfolio-full-stack/
```

## 🎯 Objetivos dos Projetos

### Beginner
- Aplicar HTML semântico
- Estilizar com CSS puro
- Adicionar interatividade básica com JavaScript

### Intermediate
- Usar layouts modernos (Flexbox, Grid)
- Consumir APIs externas
- Implementar autenticação simples
- Aprender responsividade

### Advanced
- Frameworks (React/Vue)
- State management
- Backend integration
- Deployment

## 📋 Como Contribuir

1. Criar uma branch para o projeto
2. Implementar com boas práticas
3. Documentar o processo
4. Fazer commit com mensagens claras
